What's New in 2.1.1

Here are descriptions of some of the more interesting or significant changes made to the X10 Tooling Platform for the 2.1.1 release. They are grouped into:

  • X10 Development Toolkit
  • Java back-end
  • C++ back-end
  • Additional Notes
X10 Development Toolkit
Global Search Feature X10DT now provides an X10 global search feature (Search -> Search -> X10 Search tab) that allows searching for occurrences of types, fields, and methods in a given scope.
X10 Distribution Verification The x10_platform.conf editor now verifies that the user-supplied X10 distribution location actually exists on the remote host, and performs a simple check to determine whether that directory looks like a valid X10 distribution. If not, the editor will indicate that with a message at the top of the form.

An Introduction to Programming With X10 This is a draft of the Part I of the Guide. This draft focuses on core features of the X10 language and programming model. This Guide is a work-in-progress. We anticipate that a complete draft of Part I and a substantial part of Part II will be available in early first quarter 2011. The Guide is located under the X10 Help contents.
