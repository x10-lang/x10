13  Return statement

ReturnStatement ::= return Exp? ;
Methods and closures may return values using a return statement. If the method's return type is explicitely declared void, the method must return without a value; otherwise, it must return a value of the appropriate type.
The following code illustrates returning values from a closure and a method. The return inside of closure returns from closure, not from method. 
def method(x:Int) {
  val closure = (y:Int) => {return x+y;}; 
  val res = closure(0);
  assert res == x;
  return res == x;
}
