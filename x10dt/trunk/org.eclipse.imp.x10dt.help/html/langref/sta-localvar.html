2  Local variable declaration

Short-lived variables are introduced by local variables declarations, as described in §. Local variables may be declared only within a block statement (§). The scope of a local variable declaration is the statement itself and the subsequent statements in the block. 
  if (a > 1) {
     val b = a/2;
     var c : Int = 0;
     // b and c are defined here
  }
  // b and c are not defined here.
