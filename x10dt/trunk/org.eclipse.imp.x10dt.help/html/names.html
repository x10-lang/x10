Chapter 1
Names and packages

X10 supports Java's mechanisms for names and packages [6,§ 6,§ 7], including public, protected, private and package-specific access control.
X10 supports the following naming conventions. Names of value classes should start with a lower-case leter, and those of reference classes with an upper-case letter. X10 also supports the convention that fields, local variable names and method parameter names that start with an uppercase letter are automatically considered to be annotated with final.
