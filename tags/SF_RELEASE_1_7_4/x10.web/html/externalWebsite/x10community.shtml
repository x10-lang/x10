X10 Community


    
  The following mailing lists are available for the X10 project:
  • x10-announce@lists.sf.net -- A read-only list for X10-related announcements.
  • x10-users@lists.sf.net -- A list for general X10 questions and discussion. This is also the preferred place to get help on X10. No subscription is required to post.
  • x10-developers@lists.sf.net -- A list for the X10 language evolution and development discussion, which may get quite technical.
