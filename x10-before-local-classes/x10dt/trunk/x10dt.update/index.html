X10DT Project Update Site

This Eclipse update site hosts the distribution for the X10 Development Toolkit, and all prerequisites.


The X10 project page is located at http://x10-lang.org.

Using this Update Site

Simply go to the Eclipse Update Manager (Help -> Install New Software), create a new update site and enter this site's URL:
http://dist.codehaus.org/x10/x10dt/2.0/updateSite/
