25  Casts

The cast operation may be used to cast an expression to a given type:
UnaryExpression  ::=  CastExpression
CastExpression  ::=  UnaryExpression as Type
The result of this operation is a value of the given type if the cast is permissible at run time.
The äs" operation converts the value to the given type or throws "x10.lang.ClassCastException" if the value cannot be converted. Object identity need not be preserved by the conversion. Type conversion is checked according to the rules of the Java language (e.g., [,§ 5.5]). For constrained types, both the base type and the constraint are checked. If the value cannot be cast to the appropriate type, a "ClassCastException" is thrown.
